# 概要
VRChatでの使用を想定したアウトラインにこだわりのあるシェーダーです。

オブジェクトの内側と外側とで異なるアウトラインを描画できます。

ライティングにはOpenLitを使用しています。

https://github.com/lilxyzw/OpenLit
